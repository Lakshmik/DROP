# DROP Historical Calibration Package

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP Historical Calibration Package implements the Calibration Instruments, Codes, and Quotes.


## Class Components

 * [***DepositInstrumentSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/DepositInstrumentSuite.java)
 <i>DepositInstrumentSuite</i> contains the Configuration Settings to generate the Deposit Calibration Suite.

 * [***FixFloatInstrumentSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/FixFloatInstrumentSuite.java)
 <i>FixFloatInstrumentSuite</i> contains the Configuration Settings to generate the Fix-Float Calibration Suite.

 * [***FundingCalibrationSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/FundingCalibrationSuite.java)
 <i>FundingCalibrationSuite</i> maintains the Instrument Market Quotes used in the Funding State Construction.

 * [***FundingMarkSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/FundingMarkSuite.java)
 <i>FundingMarkSuite</i> maintains the Instrument Market Quotes used in the Funding State Construction.

 * [***FuturesInstrumentSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/FuturesInstrumentSuite.java)
 <i>FuturesInstrumentSuite</i> contains the Configuration Settings to generate the Futures Calibration Suite.

 * [***GovvieInstrumentSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/GovvieInstrumentSuite.java)
 <i>GovvieInstrumentSuite</i> contains the Configuration Settings to generate the Govvie Calibration Suite.

 * [***GovvieMarkSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/GovvieMarkSuite.java)
 <i>GovvieMarkSuite</i> maintains the Instrument Market Quotes used in the Govvie State Construction.

 * [***InstrumentSuite***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/historical/calibration/InstrumentSuite.java)
 <i>InstrumentSuite</i> contains the Baseline Configuration Settings to generate the Instrument Calibration Suite.
