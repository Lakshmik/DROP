# DROP Measure PDF Package

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP Measure PDF contains Explicit R<sup>1</sup> and R<sup>d</sup> PDF's.


## Class Components

 * [***R1BernoulliDistribution***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/pdf/R1BernoulliDistribution.java)
 <i>R1BernoulliDistribution</i> implements the R<sup>1</sup> Bernoulli Probability Distribution Function.

 * [***R1DegenerateDistribution***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/pdf/R1BernoulliDistribution.java)
 <i>R1DegenerateDistribution</i> implements the Degenerate Probability Distribution Function.
