# DROP Measure Generators Package

<p align="center"><img src="https://github.com/lakshmiDRIP/DROP/blob/master/DRIP_Logo.gif?raw=true" width="100"></p>

DROP Measure Generators holds R<sup>1</sup>/R<sup>d</sup> Moment/Probability Generating Functions.


## Class Components

 * [***R1BernoulliMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1BernoulliMGF.java)
 <i>R1BernoulliMGF</i> implements the MGF of R<sup>1</sup> Bernoulli Distribution.

 * [***R1CentralChiSquaredMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1CentralChiSquaredMGF.java)
 <i>R1CentralChiSquaredMGF</i> implements the Central Chi-squared <i>R1ToR1</i> Moment Generating Function.

 * [***R1DegenerateMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1DegenerateMGF.java)
 <i>R1DegenerateMGF</i> implements the MGF of R<sup>1</sup> Degenerate Distribution.

 * [***R1MomentGeneratingFunction***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1MomentGeneratingFunction.java)
 <i>R1MomentGeneratingFunction</i> implements the <i>R1ToR1</i> Moment Generating Function.

 * [***R1NonCentralChiSquaredMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1NonCentralChiSquaredMGF.java)
 <i>R1NonCentralChiSquaredMGF</i> implements the Non-Central Chi-squared <i>R1ToR1</i> Moment Generating Function.

 * [***R1RateGammaMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1RateGammaMGF.java)
 <i>R1RateGammaMGF</i> implements the Rate Parametrized Gamma Distribution <i>R1ToR1</i> Moment Generating Function.

 * [***R1ShapeScaleGammaMGF***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/R1ShapeScaleGammaMGF.java)
 <i>R1ShapeScaleGammaMGF</i> implements the Shape/Scale Parametrized Gamma Distribution <i>R1ToR1</i> Moment Generating Function.

 * [***RdMomentGeneratingFunction***](https://github.com/lakshmiDRIP/DROP/tree/master/src/main/java/org/drip/measure/generators/RdMomentGeneratingFunction.java)
 <i>RdMomentGeneratingFunction</i> implements the <i>RdToR1</i> Moment Generating Function.
